Fitur Utama

Semua yang kamu butuhkan untuk mengelola payslip dalam satu aplikasi

Batch Lock PDF

Kunci ratusan PDF sekaligus menggunakan password dari Excel (NIK + Tanggal Lahir)

Merge PDF

Gabungkan beberapa file PDF menjadi satu dokumen dengan mudah

Split PDF (Pair Mode)

Pisahkan halaman PDF menjadi pasangan (contoh: slip depan-belakang)

Auto Rename Slip

Otomatis rename file slip gaji berdasarkan ID & Nama karyawan dari dalam PDF

Send Email Massal

Kirim payslip ke seluruh karyawan sekaligus via Gmail dengan password otomatis

Persyaratan Sistem

  • Windows 10 atau Windows 11 (64-bit)
  • Minimal 4 GB RAM (8 GB direkomendasikan)
  • Koneksi internet (hanya untuk fitur kirim email)

Catatan Penting

Windows Defender mungkin menandai file ini sebagai virus (false positive). Silakan klik "Run anyway" atau tambahkan ke exception.

Login default tersedia di dalam aplikasi (hubungi admin untuk akun baru).
